---
title: "Listar libros"
description: "Lista todos los libros con su autor, opcionalmente filtrado por género"
openapi: "GET /autores"
---

Cada libro viene con un objeto `autor` anidado (id + nombre).

## Query params

- `genero` (string, opcional): filtra por género (case-insensitive, match exacto). Ej: `ficcion`, `historia`.

## Respuesta exitosa — 200 OK


{

	"success": true,
	
	"data": [
	
		{
		
			"id": 1,
			
			"titulo": "Cien anos de soledad",
			
			"isbn": "9780307474728",
			
			"genero": "ficcion",
			
			"anio_publicacion": 1967,
			
			"autor_id": 1,
			
			"autor": { "id": 1, "nombre": "Gabriel Garcia Marquez" }
		
		}
	
	]

}

## Errores

| Status | `error.code` |
|---|---|
| 500 | `LIBROS_LIST_ERROR` |
